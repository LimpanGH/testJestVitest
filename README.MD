
# Testa Vitest (Jest finns i branchen main)


1. npm install - -save-dev  vitest





2. Create a package.json file

- Install: npm init
- Add this to package.json

```
  "scripts": {
    "test": "jest --coverage"
  },
```

3. Run: npm test
4. See testresult in terminal or open index.html in LiveServer to see result
